🛡️ Security & Safety

We believe automation should be transparent and safe. Here's why you can trust CapyOrange:

Pure Repetition: The app only repeats exactly what you record. There are no hidden "random" behaviors or background data collection. Your scripts are stored locally on your machine.

Privilege Levels (Shield Icons)

Depending on how you launch the app, you will see a different shield icon in the Script Explorer:

🛡️ ADMIN

Highest compatibility. Can interact with Task Manager, System tools, and high-privilege games. Recommended for most users.

🛡️ LIMITED

Standard user mode. Safer, but cannot send clicks/keys to any application running with Administrator rights.

🎮 Interface Guide

⏺ Record (F6)

Starts capturing your mouse and keyboard actions. Press again to save.

▶ Replay (F7)

Plays back the currently loaded script.

⏹ Stop (ESC/F10)

Emergency stop. Instantly kills any running script or recording.

⏸ Pause

Temporarily halts playback. Hit again to resume.

🗑 Clear

Clears the current list of events to start fresh.

🧙 Wizard

A step-by-step assistant to help you set up window targeting and basic recordings.

📖 Step-by-Step Tutorials

1. Create a Simple Script

  • Click Record (F6).
  • Perform the actions on your computer that you want to automate.
  • Press F6 again to finish.
  • Click Save and name your script.

2. Create a Script with Conditional Mode

  • Open the Options panel and enable Conditional Mode.
  • Capture a reference image (e.g., a "Confirm" button) using the camera icon.
  • Set the If Detected script and the Else action.
  • The macro will now only run the script if that specific image is found on your screen!

3. Create a Workflow

  • Go to the ⚡ Workflows section and click the ＋ icon.
  • Drag existing scripts from your library into the Workflow Chart.
  • Use connectors to define the sequence of execution.

4. Using Workflow Triggers

  • Add a Loop or Schedule block to your chart.
  • Place your scripts inside these blocks.
  • Configure the trigger (e.g., "Run every 10 minutes" or "Wait until 3:00 PM").
  • Click Play on the Workflow panel to activate the automation.
